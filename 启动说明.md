# 转专业OJ刷题平台 · 启动说明

## 环境要求

* Python 3.10+
* Node.js 18+

## 一键启动

### 第一步：安装依赖（仅首次）

```bash
cd backend
pip install -r requirements.txt

cd ../frontend
npm install
```

### 第二步：启动后端（终端1）

```bash
cd backend
python main.py
```

后端启动后访问：http://localhost:8000/docs 可查看API文档

### 第三步：启动前端（终端2）

```bash
cd frontend
npm run dev
```

浏览器打开：http://localhost:5173

## 使用流程

1. 打开题目列表 → 选择一道题
2. 阅读题目描述和样例
3. 在右侧Monaco编辑器中编写Python代码
4. 点击"提交代码"按钮
5. 查看评测结果：AC(通过) / WA(答案错误) / TLE(超时) / RE(运行错误)

## 评测说明

* 每组测试用例限制3秒运行时间
* 代码不能使用 os、sys、subprocess 等危险模块
* 答案对比会忽略末尾多余空白字符
